要素の分析

  • シンメトリーなレイアウトが対比構造が分かりやすくしており、一目で家の見比べであることが伝わる。
  • 背景の色面の形が家になっている。また、角が丸く柔らかい印象
  • あしらいが細く作り込まれている
  • リボンの切り込み部分の角度が広く柔らかい印象
  • リボンのあしらいが上品さ、特別感を出している
  • テキスト情報がまとまりごとに整理されている
  • 家の名前をフォントの違いによって雰囲気を伝えている
  • アイコンの使用などによって、よりカジュアルさを演出
  • 同じ視点での比較をアイコンを使って一目で分かるようにしている
  • 余白が広めにとられており、ゆったりとした印象で情報のまとまりも分かりやすい
