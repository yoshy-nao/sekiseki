もっと便利に！もっとおトクに！LINE ID 連携キャンペーン

媒体 バナー
ターゲット層 若者
キーワード 若者らしい、ポップ、カジュアル、さわやか
配色 白を基調に明度の高い色とLINEの緑でまとめられている。
フォント 細めのゴシックが使われておりすっきりとした印象
レイアウト 中心に必要な情報が配置されている。

要素の分析

  • 背景色がグラデーションになっており動きを感じる。
  • 背景色の上に重ねられているストライプがさらに動きを出している。
  • 紙吹雪が上に配置されており賑やかな雰囲気を演出。ただ、紙吹雪の色がパステル調なので、あまりうるさくない。
  • ストライプや色面の幾何学を重ねることでポップさを演出。
  • 「もっと便利に〜」のアンダーラインを破線にすることで軽やかにしている。
  • 優先度の高い数字を大きく表示している。
  • 「今なら」に吹き出しのようなあしらいをいれることで強調しよりポップに。
  • 「もれなく」の文字の上に緑色の「・」をつけて強調している。
  • 「300円」の文字のアウトラインがずれており、よりポップになっている。
  • 「（税込）」を「円」の上に置き、レイアウトを調整。
  • 手書き風の「OFF」の文字が「円」に重なっており抜け感がある。
  • 「クーポンプレゼント」のリボンの角度が強くポップさが増している。
  • リボンの背景色にストライプを入れ、より賑やかでカジュアルに。
