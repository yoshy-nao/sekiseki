【ラクスル】印刷商品 100%保証

媒体 バナー
ターゲット層 社会人、品質にこだわりのある人
キーワード 信頼感、落ち着き、シンプル
配色 青と白を基調に清潔感と信頼感を感じる配色。色数が絞られており、シンプルで落ち着いた雰囲気を出している。
フォント 細字で硬めのサンセリフ体で信頼感をイメージ。数字のフォントはやや角ばっており、落ち着いている。
レイアウト シンメトリー気味に整然と並んでおり落ち着きや堅実さを出している。

要素の分析

  • 月桂樹のあしらいは信頼をアピール
  • 「100%」という文字を大きくマーカーのあしらいで強調し、品質をアピール。
  • 「満足保証」の文字を1文字ずつ四角で囲み強調。
  • 青色の線でアウトラインを囲むことで全体を引き締めている。
  • 下の帯の文字の黒背景に白字となっている部分は、色数を増やさずに内容を強調している。
  • マーカーの黄色の強調が全体の色味が単調になりすぎることを防いでいる。
