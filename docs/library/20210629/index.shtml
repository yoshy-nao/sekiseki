媒体 バナー
ターゲット層 若者
キーワード 清潔感、シンプル、カジュアル、若者向け
配色 青を基調にした清潔感のある配色。白字も黒字も可読性のある青で、アクセントカラーの黄色も青味かかっており、全体の清潔なイメージを保っている。

要素の分析

  • 要素が少なくシンプルなのに印象的なデザインの好例
  • ロゴやテキストと容器が重なりがデザインに奥行きを出している。
  • 「キープ」の「ー」が幅いっぱいに広がっており、長時間殺菌するということを強調が強調されている。
  • 「クリア」の字を白くすることで抜け感を出している。
  • 所々に白の文字を置くことで全体が重くなるのを防いでいる。
  • 両端が揃っていて全体にまとまりがある。
