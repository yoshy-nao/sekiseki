【マネーフォワード】キャンペーンバナー

媒体 バナー
業種 IT
キーワード 楽しい、賑やか、ポップ
配色 無彩色をほとんど使わず、オレンジ色を基調にカラフルで賑やかにまとめられている。
フォント ゴシック体
レイアウト シンメトリーで情報が整理されている。

要素の分析

  • リボンを使うことでキャンペーン感や特別感を出している。
  • タイトルのドロップシャドウにやや濃いオレンジが使われているので、全体の雰囲気を保っている。
  • タイトルの「今だけ」が吹き出しになっていて、訴求力を強めて、動きを出している。
  • リボンや長方形、アイコンの角が丸いので柔らかい印象を出してる。
  • 紙吹雪の形がただの長方形だけではなくラウンドしていて、動きを感じる。
  • 背景にパターンがあり、賑やかさを強めている。
  • 下部の白の色面と要素が重なっているので、デザインに奥行きを出している。
  • 「電気料金3%OFF」と「プレミアムサービスずっと0円！」の色味を変えることで、対比関係が分かりやすい。
  • 「電気料金3%OFF」に緑を使うことで、全体の色味が単調になるのを防いでいる。
  • 数字が大きくすることで、訴求力を強めて、デザインにメリハリを出している。
