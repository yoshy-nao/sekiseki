媒体 バナー
キーワード 信頼感、フォーマル、シンプル
配色 コーポレートカラーがベースに、背景色に青みがかった薄いグレーを置くことで、全体を落ち着いた雰囲気にしており、信頼感を演出している。

要素の分析

  • レイアウトが整然としており落ち着いている。
  • それでいて、吹き出しや矢印などを使用しており堅くなり過ぎず、親しみやすい。
  • 線が細く、繊細でフォーマルな感じを出している。
  • 3つ縦に並んでいる要素のアンダーラインは長方形を感じさせ、レイアウトに安定感を出している。
  • 数字を大きくメリハリがある。
  • 両端揃え→フォーマル
