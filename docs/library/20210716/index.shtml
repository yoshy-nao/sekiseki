【吉野家】ねぎ塩豚丼

媒体 バナー
業種 飲食
キーワード さわやか、夏、カジュアル
配色 爽やかな緑を基調に配色できている。彩度の高い黄色と赤色がアクセントになっている。黒を使ってないので重くない。
フォント 若干角が丸いゴシック体が使われており、カジュアルさが演出されている。

要素の分析

  • 要素を透過させることで、爽やかで重くならないようにしている。
  • 円の中の文字が大きくはみ出しており、カジュアルさと動きを出している。
  • 緑を散らすことで、全体にバランスを生み出している。
  • 文字の大きさの差が起きい。
  • 要素が重なっている部分が多く、まとまりと奥行きを出している。
  • 白字にドロップシャドウをいれることで視認性を高め、デザインに奥行きを出している。
  • 牛丼の上からの構図によってレイアウトを安定させている。
  • 雰囲気を演出する背景が選定されている。
  • ネギが使われている健康感、みずみずしさを出している。
  • 文字数の関係上で、できる余白を要素を効果的に配置することによって埋めている。
  • 文字を斜めに配置し、動きを出している。
  • 情報の境界に罫線が効果的に用いられている。
