【LIG】優秀なエンジニアが応募したくなる会社とは

媒体 バナー
ターゲット層 ビジネスマン
キーワード 先進的、テクノロジー、シンプル、ビジネス
配色 青系統の色彩で統一されており先進的なイメージ

要素の分析

  • 背景にテクノロジーを感じさせるパターン
  • グラデーションで先進的なイメージを表現している
  • 青を基調にした配色で信頼感と先進的な雰囲気を演出。
  • フォントサイズは全体的にあまり大きくなく強弱も少ないため、落ち着いた印象を出している。
  • 要素の数は少なく、要素を余白を大きめに配置することで、すっきりとした雰囲気を出している。
  • レイアウトはシンメトリー気味で要素を整列させることで落ち着いた印象。
  • あしらいも最小限でポップな雰囲気を出さないようにしている。
