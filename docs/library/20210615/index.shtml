【レンズモード】15周年キャンペーン

媒体 バナー
ターゲット層 幅広い年代（20代〜50代）の女性。
キーワード カジュアル、女性らしい、シンプル、キャンペーン
配色 白を基調に暖色系でシンプルにまとめられている。

要素の分析

  • グラデーションにより、全体を明るく動きのある感じにしている。
  • 数字が大きく立体的なあしらいにより、優先度の高い数字が強調されている。
  • 大感謝祭の文字のサイズを変え、ベースラインを上下にずらすことでリズムを出している。
  • 金色のあしらいは賑やかさや豪華さを演出し、15周年の感謝祭の特別な雰囲気を表現している。
  • 斜めの図形は単調なレイアウトになるのを防ぎ、画面を区切ることで全体を引き締めている。
  • 女性の写真は人々を呼び込むポーズで引きがある。若い女性を選定することで親しみのある雰囲気にしている。
  • リボンの帯は15周年の特別感を演出。
